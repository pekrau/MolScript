Software libraries used in MolScript

GLUT library

  • http://www.opengl.org/resources/libraries/glut/ provides the distribution for the GLUT library (The OpenGL Utility Toolkit) originally written by Mark Kilgard. Version 3.5 or later is required.
  • See also "OpenGL Programming for the X Window System", the book Mark Kilgard has written about OpenGL and the X window system.

JPEG library

  • http://www.ijg.org/ is the official website for the JPEG library from the Independent JPEG Group (IJG). Version 6a or later is required.

PNG library

  • http://www.libpng.org/pub/png/ is the website for the PNG library. Version 0.96 or later is required. The zlib library may also be obtained at this location.

GIF library

  • http://www.libgd.org/ is the current site for the gd library (a C API for GIF creation), originally written by Thomas Boutell. Version 1.3 or later should be used. This version does not use LZW compression, so it avoids the legal problems with the Unisys patent on that algorithm.
  • Version 1.2 of gd can be also be used with MolScript without modification, but since this version of gd uses LZW compression, it is not recommended for legal reasons.

Other relevant sites

Raster3D

  • The Raster3D Website, with distribution information. Version 2.4b or later is required for use with MolScript v2.1.
  • Ethan A. Merritt, author (with others) and maintainer of Raster3D.

OpenGL

  • The OpenGL Organization website.
  • The OpenGL Center at SGI.
  • The Mesa 3D Graphics Library, originally by Brian Paul: the OpenGL look-alike 3D-rendering software library.

PostScript

  • PostScript, at the official Adobe Inc. website.
  • Ghostscript, Ghostview and GSview, PostScript interpreter software.

VRML 2.0

  • VRML Virtual Reality Modeling Language.
  • The VRML97 specification, the ISO codification of VRML 2.0.

SGI (aka RGB) image file

  • The specification of the SGI image file format

GNU software

  • The GNU Project.

Protein 3D structure data

  • The Protein Data Bank (PDB) at RSCB. This is the scientific data bank where published protein 3D coordinate data sets are deposited by the scientists and from where the data sets can be retrieved by anyone. The specification of the PDB coordinate file format is available. The 3D data sets can be searched in different ways.
